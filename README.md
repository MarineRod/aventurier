# Déplacement d'un personnage dans une grille

## Description

Cette application Java simule un **personnage qui se déplace sur une carte** avec des obstacles.  
La carte est représentée par une **grille 2D**, où les cases libres et les obstacles sont indiqués respectivement par ` ` et `#`.  
Le personnage se déplace selon une suite de directions cardinales (N, S, E, O) et ne peut pas traverser les obstacles ou sortir de la carte.

---

## Fonctionnalités

- Déplacement du personnage dans 4 directions :  
  - N = Nord (haut)  
  - S = Sud (bas)  
  - E = Est (droite)  
  - O = Ouest (gauche)
- Affichage de la carte dans la console
- Affichage de la position du personnage après chaque déplacement
- Validation automatique des mouvements (ne dépasse pas les limites de la carte)
